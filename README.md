## 综述

**总有更值得做的事。**
本仓库的目的就在于让读者用最少的时间解决整体上的问题，了解有哪些细节可能出问题，不将宝贵的时间用于在A4纸上雕花。

如果对您有帮助，请不吝您的``Star``，这对于项目的传播和我本人都很重要，谢谢！

* 该项目为个人维护的开源项目，目前内容包括但不限于山东大学信息科学与工程学院课程的相关资料。

* 不应当随意删除本项目的文件。任何资料都有存档的价值。
